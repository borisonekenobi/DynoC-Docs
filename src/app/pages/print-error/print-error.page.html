print_error()

Error: The print_error() function cannot print numbers. Integers and floats are currently sent to stdout instead of stderr. You can track progress on this issue or report bugs on the GitHub Issues page.

Writes one or more values to standard error output. This is useful for displaying errors separately from normal program output.

Syntax

print_error(value1, value2, ...);

Examples

print_error("Error: invalid input");
