Getting Started

Welcome to DynoC! This page will walk you through installing the compiler and running your first program.

1. Download the Compiler

Info: For now, DynoC compiles programs for Linux and Windows x86-64. Other targets will be supported in the future. You can track progress on this issue or report bugs on the GitHub Issues page.

You can find the latest release of the DynoC compiler on the GitHub Releases page. Download the version that matches your operating system.

2. Write Your First Program

Create a file called hello.dc with the following code:

print_line("Hello, DynoC!");

3. Compile

Open a terminal in the folder containing hello.dc and run: 

dynoc hello.dc

An executable file should be created, with the following output: 

Successfully compiled to ...

4. Run the Program

Run the executable file, and you should see the output: 

Hello, DynoC!
