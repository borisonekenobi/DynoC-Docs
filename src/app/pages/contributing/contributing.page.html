Contributing

The DynoC compiler is currently a closed-source project. For now, contributions are limited to providing feedback, reporting bugs, and suggesting new features through the GitHub Issues page.

If and when the compiler becomes open source, anyone will be able to fork the repository, make improvements, and submit a pull request for review. Until then, your input through issues and discussions is highly appreciated and will help shape the future of the language.

Note: If you encounter a bug, please include steps to reproduce, expected behavior, and your environment details (OS, version, etc.). This will make it easier to address issues quickly.

Reporting Issues

Found a bug? Please report it on the GitHub Issues page.

Feature Requests

Have an idea for a new feature? Open a feature request on GitHub.

Code of Conduct

Please be respectful and constructive. We want the DynoC community to be welcoming to everyone.
