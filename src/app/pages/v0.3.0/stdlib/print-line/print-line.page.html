print_line()

The print_line() function outputs text or values to the standard output (usually the console) and adds a newline at the end.

It behaves like print(), but automatically moves the cursor to the next line after printing.

This is the most common way to display messages or results.

Syntax

(value);
(value1, value2, ...);
Parameter Type Description
value any The value or expression to print.

Description

print_line() converts each argument to a string and writes them to the output stream, followed by a newline character (\n).

This function is useful for printing entire lines or structured output without manually adding "\n".

Notes

  • Automatically appends a newline (\n) after all output.

  • Values are separated by spaces when multiple arguments are passed.

  • To print without a newline, use print().

  • For errors, use print_error().

See Also
