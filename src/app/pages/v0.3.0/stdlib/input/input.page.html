input()

Warning: The input() function does not handle multi-line paste correctly. If multiple lines are pasted into the terminal, they are treated as a single input instead of being split. Please see the GitHub Issues page for more details.

Reads a line of text from standard input and returns it as a string. Input is terminated when the user presses Enter.

Syntax

var text = input();

Examples

print("Enter your name: ");
var name = input();
print_line("Hello, " + name + "!");
