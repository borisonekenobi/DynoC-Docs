input()

The input() function reads a line of text from standard input (stdin).

It optionally displays a prompt before waiting for user input.

The input is always returned as a string.

Syntax

();
(prompt);
Parameter Type Description
prompt string (optional) Text to display before waiting for input.
Returns Type Description
value string The text entered by the user.

Description

input() pauses program execution and waits for the user to type a line of text, ending with the Enter key.

The entered text is returned as a string without the newline character.

If a prompt is given, it is displayed before reading input.

Notes

  • Always returns a string, regardless of what the user types.

  • Works well with print() and print_line() for interactive programs.

  • Input reading blocks execution until the user presses Enter.
