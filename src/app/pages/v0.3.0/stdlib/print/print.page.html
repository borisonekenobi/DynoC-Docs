print()

The print() function outputs text or values to the standard output (usually the console) without adding a newline at the end.

It can print any value, including numbers, strings, booleans, arrays, and characters.

Multiple values can be printed in one call, separated by commas.

Syntax

(value);
(value1, value2, ...);
Parameter Type Description
value any The value or expression to print.

Description

print() converts each argument to its string form and writes it directly to the output stream.

Unlike print_line(), it does not move the cursor to a new line after printing.

This makes it useful for inline output or building strings across multiple calls.

Notes

  • print() automatically converts non-string values to strings.

  • To add a newline at the end, use print_line() .

  • To print error messages, use print_error() , which writes to the error stream.

See Also
