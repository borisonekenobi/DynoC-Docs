print_error()

The print_error() function outputs text or values to the standard error stream (stderr).

It works like print(), but is intended for error messages, warnings, or diagnostic output.

Using print_error() helps separate normal program output from error or debug information.

Error: The print_error() function cannot print numbers. Integers and floats are currently sent to stdout instead of stderr. You can track progress on this issue or report bugs on the GitHub Issues page.

Syntax

(value);
(value1, value2, ...);
Parameter Type Description
value any The value or expression to print.

Description

print_error() writes its output to the error stream instead of the standard output stream.

Output written to stderr is not mixed with normal program output, which makes it easier to log or redirect separately.

Notes

  • Sends output to stderr instead of stdout.

  • Useful for logging, warnings, and errors.

  • Does not affect normal print() or print_line() output.

  • Can be redirected separately from normal output in most environments.
