Operators

Operators in DynoC are symbols that perform operations on values or variables.

DynoC supports a wide range of unary and binary operators similar to C++ and Java.

Arithmetic Operators

Arithmetic operators perform basic math on numeric types.

Operator Description Example Result
+ Addition 3 + 2 5
- Subtraction 5 - 1 4
* Multiplication 4 * 2 8
/ Division 8 / 2 4
% Modulo (remainder) 9 % 4 1
Info: Division between two int values results in an int. Use float for decimal results.

Unary Operators

Unary operators act on a single operand.

Operator Description Example Result
+ Unary plus +x x
- Negation -x Negative value
++ Increment (pre/post) ++x, x++ Increase by 1
-- Decrement (pre/post) --x, x-- Decrease by 1
! Logical NOT !true false
~ Bitwise NOT ~5 Bitwise complement

Comparison Operators

Comparison operators return a bool (true or false).

Operator Description Example Result
== Equal to a == b true if equal
!= Not equal to a != b true if not equal
< Less than a < b true if less
<= Less or equal a <= b true if less or equal
> Greater than a > b true if greater
>= Greater or equal a >= b true if greater or equal

Logical Operators

Combine or invert boolean values.

Operator Description Example Result
&& Logical AND a && b true if both are true
|| Logical OR a || b true if either is true
! Logical NOT !a Inverts boolean value

Bitwise Operators

Operate on binary representations of integers.

Operator Description Example Result
& Bitwise AND a & b Bitwise AND
| Bitwise OR a | b Bitwise OR
^ Bitwise XOR a ^ b Bitwise XOR
~ Bitwise NOT ~a Bitwise complement
<< Left shift a << 1 Multiply by 2
>> Right shift a >> 1 Divide by 2
