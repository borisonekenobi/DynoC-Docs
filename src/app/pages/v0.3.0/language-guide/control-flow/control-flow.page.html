Control Flow

Control flow statements let you change how your program runs by making decisions and repeating actions.

DynoC supports standard conditional and looping structures similar to C++ and Java.

If / Else

Use if to run code only when a condition is true.

You can chain multiple conditions with else if, and handle all remaining cases with else.

 a = 10;

 (a > 10)
{
    (Greater than 10);
}
  (a == 10)
{
    (Equal to 10);
}

{
    (Less than 10);
}

While Loop

A while loop repeats as long as its condition is true.

 i = 0;

 (i < 5)
{
    (i);
    i++;
}

For Loop

A for loop is useful when you know how many times you want to run something. 
 ( i = 0; i < 5; i++)
{
    (i);
}

You can also omit parts of the loop if not needed:

 i = 0;
 (; i < 5;)
{
    (i);
    i++;
}

Break and Continue

Use break to exit a loop early, and continue to skip to the next iteration.

 ( i = 0; i < 10; i++)
{
     (i == 5) ;        // stop the loop
     (i % 2 == 0) ; // skip even numbers
    (i);
}

Exit

Use exit to stop the entire program immediately.

You can pass a value to indicate an exit code.

 (has_error)
{
    (Something went wrong!);
     1;
}
