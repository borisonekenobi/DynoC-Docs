Control Flow

DynoC supports standard control flow constructs, similar to C-like languages.

If / Else

var x = 10;

if (x > 5) {
  print("Greater than 5");
} else {
  print("5 or less");
}

While Loop

var i = 0;
while (i < 5) {
  print(i);
  i++;
}

For Loop

for (var i = 0; i < 3; i++) {
  print(i);
}

Break / Continue

for (var i = 0; i < 5; i++) {
  if (i == 2) continue;
  if (i == 4) break;
  print(i);
}
// prints: 0, 1, 3
