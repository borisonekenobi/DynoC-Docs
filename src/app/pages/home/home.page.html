DynoC Programming Language

A modern, lightweight programming language with C-like syntax and dynamic typing.

Get Started View on GitHub

Why DynoC?

  • C-style syntax with dynamic typing for fast prototyping.
  • Simple runtime that compiles to efficient assembly code.
  • Cross-platform support planned (Linux, Windows, 32/64-bit).
  • Growing standard library with familiar functions like print() and input().
