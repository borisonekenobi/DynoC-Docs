Examples

Here are some basic DynoC programs to help you get started.

Hello World


print_line("Hello, world!");

Variables and Math


var x = 10;

Control Flow


var n = 5;
if (n > 0) {
    print_line("Positive");
} else {
    print_line("Non-positive");
}

Functions


int square(int n) {
    return n * n
}

print_line("Square of 4:", square(4));
